# recipe-restAPI

## General info
Recipe REST-api written in django. It includes:
- docker
- unittest
- DRF
- postgres db
- github CI


## Setup
To run this project, run following command:

```
$ docker-compose build --no-cache
```

In case of any problems try:
```
$ docker-compose up -d --force-recreate
```

In order to run migrations:
```
$ docker-compose run app python manage.py migrate
```

Optionally create super user in a running container:
```
$ docker-compose run app python manage.py createsuperuser
```

## Documentation
When container is running, documentation generated by swagger is available [here](http://localhost:8000/docs/).
